# Rahul Global Institute of Information Technology — Official Website

> A fully responsive college website built with pure HTML, CSS, and JavaScript.

## 🔗 Live Demo

https://animated-dragon-76cfd2.netlify.app/

---

## 📌 Features

- Responsive design (mobile + desktop)
- Hero image slider with auto-play
- Breaking news ticker
- Notices & PDF upload section
- Research publications listing
- Media coverage section
- Upcoming events sidebar
- **Student Contact / Enquiry Form** (Name, Email, Mobile, Message)
- **Admin Panel** (password protected) to manage:
  - News & Updates
  - Media Coverage
  - Research Publications
  - Events
  - Notices / PDF uploads
  - Ticker messages
  - Student messages / enquiries

---

## 🔐 Admin Access

Go to the website → click **⚙ Admin** (top-right) → login with credentials.

> Credentials are stored in `index.html` inside the `ADMINS` array (line ~894). Change them before deploying publicly.

---

## 🚀 How to Deploy

### Option 1 — GitHub Pages (Free)
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Select branch `main` → Save
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

### Option 2 — Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Drag & drop this project folder
3. Get your live link instantly

### Option 3 — Vercel (Free)
1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repo
3. Deploy in one click
